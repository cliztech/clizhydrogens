# Brand Checklist

Use this checklist before submitting any PR involving UI or Content changes.

## Visuals
- [ ] **Colors**: Accents are #FF00FF (Pink) or #00FFFF (Cyan). No unapproved colors.
- [ ] **Contrast**: Text contrast ratio meets WCAG AA (4.5:1). (Black on Neon = OK).
- [ ] **Borders**: Elements have hard black borders (1px/2px). No drop shadows.
- [ ] **Typography**: Headings are Space Grotesk (Caps). Body is Courier Prime.

## Content & Tone
- [ ] **Voice**: Is it cheeky/humorous? (Avoid corporate speak).
- [ ] **Formatting**: No widows/orphans in headings.
- [ ] **Images**: Alt text included and descriptive (on-brand).

## Technical
- [ ] **Mobile**: Layout works on 320px width.
- [ ] **Fonts**: Webfonts loaded correctly (no FOUT).
