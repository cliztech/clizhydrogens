# SEO Checklist

## Page Level
- [ ] **Title Tag**: Exists, unique, < 60 chars.
- [ ] **Meta Description**: Exists, unique, < 160 chars.
- [ ] **H1**: Only one H1 per page.
- [ ] **URL**: Clean slug, no query params for main content.

## Structured Data
- [ ] **JSON-LD**: Valid schema markup present (Test with Google Rich Results Test).
- [ ] **Breadcrumbs**: Properly marked up.

## Assets
- [ ] **Images**: Next-gen format (WebP/AVIF), proper sizing, Alt text present.
- [ ] **Lazy Loading**: Below-fold images lazy loaded.

## Social
- [ ] **Open Graph**: Image and Text appear correctly in preview.
