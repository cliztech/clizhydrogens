# UX Guidelines

## 1. Core Principles
- **Brutalist Functionality**: The raw aesthetic should not impede usability. Buttons must look clickable (via borders/hover states).
- **Mobile First**: Design for the smallest screen (320px) first.
- **Speed**: Interactions should feel instant.

## 2. Navigation
- **Hamburger Menu**: Use a large, blocky full-screen overlay for mobile menu.
- **Sticky Header**: Keep cart and logo visible.

## 3. Product Page (PDP)
- **Buy Box**: prominent "ADD TO CART" button (Neon background).
- **Images**: Large, uncropped images. Grid layout.
- **Specs**: Monospace lists for product details.

## 4. Cart
- **Drawer**: Slide-out drawer preferred over separate page.
- **Upsell**: "Cheeky" upsell prompts in the cart (e.g., "Don't be cheap, add a sticker").
